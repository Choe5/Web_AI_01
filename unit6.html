01

機器學習基礎

Machine Learning
機器學習 Machine Learning / ML

透過資料訓練模型，讓電腦系統自動學習與改進，不需人工編寫死板的規則。

深度學習 Deep Learning

模仿大腦神經元的「多層神經網絡」技術，特別擅長處理影像與語音等複雜資料。

神經網絡 Neural Network

模擬人類大腦神經元連接結構的人工架構。

監督學習 Supervised Learning

使用「有標準答案（標籤）」的資料來訓練模型。例如：給每張圖片標註「貓」或「狗」。

非監督學習 Unsupervised Learning

使用「沒有標記」的資料，讓模型自行尋找其中的規律與結構。

強化學習 Reinforcement Learning

透過獎勵與懲罰機制，讓 AI 在環境中不斷嘗試，學習最佳的行為策略。

過擬合 Overfitting

AI 讀死書。模型把訓練資料的「雜訊」也背下來了，遇到新資料就失靈。

欠擬合 Underfitting

AI 不讀書。模型太簡單，連訓練資料的基本規律都學不會。

泛化 Generalization

AI「舉一反三」的能力。不只會做課本題目，還能應付現實中的各種變化。

正則化 Regularization

強迫 AI「抓重點」。在訓練時加入懲罰機制，不讓模型鑽牛角尖去背細節。

提前停止 Early Stopping

見好就收。在 AI 開始死背之前，就趕快結束訓練。

資料增強 Data Augmentation

把照片旋轉、修剪或變色，讓 AI 看過同一張圖的各種變體，防止它死背單張圖片。

02

大語言模型與對話核心

LLM & Context
大型語言模型 LLM

如 GPT 系列，經過巨量文字訓練，能理解與生成人類語言的模型。

字詞單位 Token

AI 處理文字的最小碎片（可能是一個字或字根）。它是計算成本、速度與記憶量的基準。

上下文視窗 Context Window

AI 單次對話能「記住」的訊息範圍。超出此範圍，AI 會開始遺忘先前的內容。

提示詞工程 Prompt Engineering

透過設計精確的輸入指令，引導 AI 產出更理想、更專業的答案。

AI 幻覺 AI Hallucination

指 AI 以自信的語氣產生看似合理、但事實完全錯誤或虛構的內容。

03

自然語言處理

NLP
分詞 Tokenization

將整段文字拆解成 Token 的過程，是所有 NLP 任務的起點。

轉換器架構 Transformer

現代 AI 的底層架構，靠「自注意力機制」理解文字間的深層邏輯。GPT 和 BERT 都基於此。

雙向編碼理解模型 BERT

擅長「理解」語意，會同時讀取文字的前後文。

生成式預訓練轉換器 GPT

擅長「生成」內容，能像人類一樣流利地寫作與對談。

少樣本學習 Few-shot Learning

只要給 AI 幾個範例，它就能學會新的任務。

04

電腦視覺與生成技術

Computer Vision
圖像分類 Image Classification

辨識圖片中的主角是誰（例如：這是一隻貓）。

物體偵測 Object Detection

找出圖片中所有物件的位置，並用框框標註出來。

圖像分割 Image Segmentation

將圖片分割成不同區域，精確到像素級的分類。

臉部辨識 Face Recognition

辨識並驗證圖像中人臉的身份。

生成式人工智慧 Generative AI

能夠創造新內容（文字、圖像、音樂等）的技術，如 GPT、Midjourney。

擴散模型 Diffusion Model

透過「去噪」技術生成高品質影像的技術（如 Midjourney 採用的原理）。

05

部署、應用與倫理

Deployment & Agents
AI 代理 AI Agent

具備「自主性」的系統，能自行規劃步驟並調用工具（如發郵件、查資料）來達成任務。

模型情境協議 MCP（Model Context Protocol）

讓 AI 能連接外部工具（如查天氣、讀檔案、查日曆）的共同規格。就像「AI 的 USB-C 接口」—插上就能用，不需要每次重新設定。

代理技能 Agent Skills

AI Agent 能執行的一項具體能力，例如「搜尋網路」、「寄出 Email」、「讀取檔案」。技能越多，Agent 能完成的任務就越複雜，就像員工學的技能越多能接的工作就越多。

本地模型 Local Model

運行在自己電腦設備上的模型，隱私度高且無須連網。

雲端模型 Cloud Model

透過 API 串接廠商伺服器（如 GPT-4），性能強大但需依賴網路。

開源模型 Open Source

開發者公開模型的原始碼、架構以及模型權重，任何人都可以下載、部署、修改。

閉源模型 Closed Source

模型受專利保護，只能透過 API 或網頁介面使用，無法看到內部運作邏輯。

種子 Seed

初始化隨機數生成器的數值。設定相同種子可確保在相同條件下產出完全一致的結果。

檢索增強生成 RAG (Retrieval-Augmented Generation)

讓 AI 先去「翻閱」特定文件或資料庫，再根據結果回答，是解決幻覺的關鍵。

應用程式介面 API

軟體之間溝通的橋樑，讓你將 AI 功能嵌入到自己的 App 中。

推論 Inference

模型實際根據輸入產生結果的運作過程。

微調 Fine-tuning

針對特定領域資料進行二次訓練，讓模型變成專才。

深度偽造 Deepfake

使用 AI 合成極其逼真的影像或聲音，常涉及倫理與詐騙疑慮。

邊緣計算 Edge Computing

在靠近資料源的設備（如手機）進行運算，減少延遲。

偏見 Bias

AI 因為訓練資料不均，而產生的不公平或歧視性決策。

06

進階模型機制與架構

Advanced Architectures
多模態 Multimodal

指 AI 能同時處理並理解多種型態的資料（文字、影像、語音、影片），並在不同型態間轉換。

參數 Parameters

模型內部的變數（權重與偏差），代表模型的「腦容量」。通常參數越多，能力越強。

自注意力機制 Self-Attention

Transformer 的靈魂。讓模型在處理文字時，能判斷哪些詞對理解當前語義最重要。

混合專家模型 MoE

模型由多個「專家」小組組成，處理問題時只激活部分專家，節省運算資源（如 GPT-4 傳聞採用）。

07

訓練與優化技術

Training & Alignment
預訓練 Pre-training

在海量通用資料上進行的大規模初步訓練，賦予 AI 基礎的語言與常識能力。

人類回饋強化學習 RLHF

透過人類對 AI 回答進行評分，引導 AI 行為更符合人類的價值觀、更有禮貌且更安全。

量化 Quantization

將高精度數值壓縮成低精度，大幅降低對顯示卡記憶體（VRAM）的需求，讓普通電腦也能跑大模型。

LoRA Low-Rank Adaptation

高效率微調技術。不需要改動整個模型，只訓練極小比例的附加參數，就能讓模型學會特定風格。

08

提示詞進階策略

Advanced Prompting
思維鏈 Chain of Thought / CoT

誘導 AI「一步一步思考」的技術。要求 AI 在給出答案前先寫出推理過程，顯著提升邏輯問題能力。

少樣本／零樣本提示 Few/Zero-shot

「少樣本」是給 AI 幾個範例；「零樣本」則是直接下令。考驗模型的通用理解能力。

系統提示詞 System Prompt

設定 AI 的「人格」或「底層規則」。在對話開始前就餵給 AI 的指令（例如：你現在是一位資深工程師）。

09

未來趨勢與通用人工智慧

AGI & Future
通用人工智慧 AGI

AI 發展的終極目標：在各個認知領域都能達到或超越人類水平，具備自我學習與跨域解決問題的能力。

LLM as OS 大語言模型作業系統

未來的電腦操作不再是點擊圖標，而是由 LLM 作為核心調度所有軟體工具來完成任務。

具身智慧 Embodied AI

將 AI 大腦放入物理實體（如人形機器人），讓 AI 能在現實世界中感知、移動並操作物體。
